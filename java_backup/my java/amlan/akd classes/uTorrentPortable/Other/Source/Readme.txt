ABOUT UTORRENT PORTABLE
=======================
The uTorrent Portable Launcher allows you to run uTorrent from a removable drive whose
letter changes as you move it to another computer.  The application can be entirely self-
contained on the drive and then used on any Windows computer.

INSTALLATION / DIRECTORY STRUCTURE
==================================
By default, the program expects the following directory structure:

-\ <--- Directory with uTorrentPortable.exe
	+\App\
		+\uTorrent\
	+\Data\
		+\settings\
