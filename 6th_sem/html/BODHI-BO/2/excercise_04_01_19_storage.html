STORAGE

Storage refers to holding information somewhere. RAM, Random Acess Memory, Short term memory. It is volatile memory because the memory is automatically "erased" when the power is turned off or interrupted. The RAM is located inside the computer case on the motherboard. The motherboard is not the keyboard. The keyboard is what we type with. The motherboard holds RAM memory, electronic circuit and other computer parts including the central processing unit. ROM, Read Only Memory is not volatile meaing the memory is still there when the power is interrupted or turned off. When the computer is turned back on again, ROM memory is still in storage of the internal hard disk.
